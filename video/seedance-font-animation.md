# Seedance 2.0 字体开场动画提示词逻辑

> 来源：焱公子 抖音教学截图
> 适用：Seedance 2.0 字体动画开场

## 核心逻辑

### 字体层设计
```
主体文字：
- 字体风格：刚劲楷书
- 颜色：银红色
- 肌理：刀锋划过的锐利肌理 + 浪涛翻涌的动态笔触
- 边缘效果：暖橙色辉光晕染

辅助文字（英文）：
- 字体：衬线体
- 作用：形成中英双语层级排版
```

### 背景层设计
```
基础氛围：
- 色调：墨绿色（沉郁厚重）
- 肌理：颗粒感 + 哑光质感
- 景深：远景虚化处理

意象元素（负形融入）：
- 山峦剪影
- 战旗飘动
- 人物剪影
- 血色光斑点缀（强化情绪）
```

### 动画时长设计
```
12-15秒结构：
- 定格终局画面
- 字体动态特效
- 色彩校正过渡
- 镜头语言运用
```

## 提示词模板

### 基础版
```

背景设计：
墨绿色[主题场景]远景，颗粒肌理感，虚化的[场景元素]以低饱和度负形效果融入背景，沉郁厚重的氛围，零星血色光斑点缀

构图：
中心聚焦式布局，中英双语层级排版，明暗对比强烈
```

```
字体动画开场，12-15秒：

【字体层】
主体：[文字内容]
- 风格：刚劲楷书 + 锐利肌理 + 动态笔触
- 色彩：银红色主体 + 暖橙色辉光边缘
- 英文副标：衬线体，层级排版

【背景层】
- 主色调：墨绿色沉郁氛围
- 肌理：颗粒感 + 哑光质感
- 意象：[主题元素]虚化融入，低饱和度
- 点缀：血色光斑强化情绪

【动画参数】
- 时长：12-15秒
- 节奏：定格终局
- 特效：动态字体 + 色彩过渡
- 镜头：[推/拉/摇/移]
```

## 关键参数对照表

| 参数 | 描述 | 示例 |
|------|------|------|
| 字体风格 | 刚劲楷书/行书/草书 | 刚劲楷书 |
| 字体颜色 | 主体色+边缘晕染色 | 银红色+暖橙色 |
| 肌理效果 | 刀锋/浪涛/金属/裂纹 | 刀锋划过 |
| 背景色调 | 主题情绪色 | 墨绿色沉郁 |
| 景深处理 | 远景/中景/近景 | 远景虚化 |
| 意象元素 | 主题相关符号 | 山峦/战旗/剪影 |
| 动画时长 | 8-15秒 | 12-15秒 |

## 应用案例

### 案例1：武侠主题
```
字体：侠骨柔情，银红色刚劲楷书，刀锋肌理
背景：墨绿色江湖远景，战旗飘动，血色光斑
时长：12秒定格终局
```

### 案例2：科幻主题
```
字体：星际征途，银蓝色刚劲字体，电光肌理
背景：深蓝太空远景，星舰剪影，粒子光斑
时长：15秒推进镜头
```

### 案例3：古风主题
```
字体：长安月下，金色楷书，流光肌理
背景：墨蓝夜空远景，楼阁剪影，萤火点缀
时长：10秒淡入定格
```

---

## 注意事项

1. **字体选择**：楷书适合武侠/古风，无衬线体适合现代/科幻
2. **颜色搭配**：主体色与背景形成对比，边缘晕染色增加层次
3. **肌理统一**：字体肌理与主题情绪匹配
4. **动画节奏**：字体动画不宜过快，保持可读性
5. **中英搭配**：英文用衬线体，形成视觉层级
